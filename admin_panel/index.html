Setup Instructions
1. Push your project to a GitHub repository.
2. Open backend/routers/update.py — set GITHUB_REPO = "yourusername/yourrepo".
3. On the server, clone the repo: git clone https://github.com/yourusername/yourrepo.git
4. Run: pm2 start ecosystem.config.js
5. Clicking "Apply Update" runs git pull --rebase and restarts the server.
